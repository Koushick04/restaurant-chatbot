# Contributing

Thanks for improving Restaurant Chatbot.

## Local Setup

1. Install Node.js 18 or newer.
2. Install backend dependencies from `backend/` with `npm install`.
3. Install frontend dependencies from `frontend/` with `npm install`.
4. Copy `.env.example` to `backend/.env` and `frontend/.env`.
5. Start the backend with `npm run dev`.
6. Start the frontend with `npm run dev`.

## Development Guidelines

- Keep secrets out of commits.
- Keep frontend code typed and responsive.
- Keep API responses consistent with `docs/API.md`.
- Update docs when behavior, setup, or environment variables change.
- Run `npm run build` in `frontend/` before opening a pull request.

## Commit Style

Use short, descriptive commit messages, for example:

```text
Add reservation export controls
Fix document upload validation
Update deployment guide
```
